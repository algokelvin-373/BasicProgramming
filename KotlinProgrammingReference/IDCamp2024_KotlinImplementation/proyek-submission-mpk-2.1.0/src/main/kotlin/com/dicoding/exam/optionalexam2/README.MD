#### Contoh 1:

- Keluaran: `11`

#### Contoh 2:

- Masukan: `number` = `10`
- Keluaran: `1`
- Penjelasan:
    - Digit terkecilnya adalah `0` dan digit terbesar adalah `1`.
    - Hasil perjumlahan dari `0 + 1` adalah `1`
